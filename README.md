# HHI Language Licensing

This repository defines how Hollow House Institute governance language
is licensed for commercial use.

HHI terminology is not descriptive branding.
It is operational governance infrastructure.

If your organization uses these terms to shape decisions, audits,
policies, or products, you are using governed intellectual property.
## Authority & Dependency Chain

This repository is part of the Hollow House Institute governance stack.

**Authoritative order:**
1. Hollow_House_Institute — Root doctrine and institutional authority
2. Hollow_House_Standards_Library — Canonical definitions and terminology
3. HHI-GOV-01 — Governance standard and structural requirements
4. HHI_LUL_01 — Language and terminology licensing
5. Master_License_Suite — Legal enforcement and usage constraints

Definitions, terms, and permitted usage are governed upstream.
Downstream use must not contradict or reinterpret upstream authority.
## Licensing Tiers
- Research / Academic: Free with attribution
- Internal Enterprise: Annual license
- Vendors / Consultancies: Per-client license
- Tooling / SaaS: Embedded license
## Commercial & Institutional Use

Commercial or institutional use of Hollow House Institute terminology
(including **Behavioral Drift**, **Decision Substitution**, **Stable Mirrors**,
**Governance Infrastructure**, and related terms) requires a language use license.
